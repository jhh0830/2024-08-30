문단을 나누는 태그

문단 영역을 나누는 태그로는 p 태그와 pre 태그가 있다. 단, p 태그는 줄바꿈 입력을 별도의 태그로 지정해야 한다.
그리고 공백은 한개의 공백만을 표시하기 때문에 여러개의 공백을 한번에 표현하고자 한다면          기호 문구를 작성해야한다. 

            pre 태그는 여러           공백이 인식 가능하고,
            줄 바꿈 등을 포함하여 입력한 내용 그대로를 문단으로 표현해준다.

그 밖에 글자를 다루는 태그들

일반 골꼴 (태그로 감싸 지지 않은 텍스트)

b: 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그

em : 글자를 기울여서 표시하는 태그

i : 글자를 기울여서 표시하는 태그

mark : 형광펜 효과를 주는 태그

u : 글자에 밑줄이 그어지는 태그

s : 글자에 취소선을 넣어주는 태그

strike : 글자에 취소선을 넣어주는 태그

small : 글자를 작게 표현해주는 태그

big : 글자를 크게 표현해주는 태그

IOT 란 각종 사물에 센서와 통신 기능을 내장해 인터넷에 연결하는 기술이다.

sub : 기본 글자에 아래 첨자 를 나타내는 태그

sup : 윗첨자 를 나타내는 태그가

글자 관련 태그 응용

태그들은 해당 태그 내에서 중첩으로 사용 가능하다. 굵은글자들을 사용할 수 도 있고, 기울이거나,취소선을 넣는 등 다양하게 사용할수있다.
형광펜을 넣어 글자를 강조할수있다.
