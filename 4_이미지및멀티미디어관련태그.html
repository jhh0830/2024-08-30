이미지 관련 태그

* src, alt 속성



* src
- 사진의 경로를 기술하는 부분
- 경로를 잘못 지정하면 엑박이 뜸!!

* alt
- 사진에 대한 설명을 기술하는 부분
- 사진에 경로가 잘못되었거나 이미지를 제대로 표현할 수 없을 경우 대체 텍스트의 용도
- 시작장애인을 위한 스크린리더 (화면낭독기) 에서 이미지를 읽어주는 설명 문구로도 활용 가능 !! (접근성 : 모든 사람이 평등하게 웹 사이트를 이용 가능한 특징)

* width, height 속성

이미지의 가로, 세로길이를 조정 가능
고정길이 (px), 가변길이 (%) 로 지정

- 고정길이단위 (px) 화면사이즈가 조정되더라도 이미지의 크기는 변동이 없음 (기본값)



- 가변길이 단위 (%) 화면사이즈 혹은 부모요소의 사이즈에 따라 이미지의 크기도 같이 변동 >
